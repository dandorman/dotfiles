# Dan's Dot Files

## Italics in iTerm

1. Run `tic xterm-256color-italic.terminfo` from this directory.
2. In iTerm:
  a. Visit the “Terminal” tab of a given theme.
  b. Fill in “Report Terminal Type:” with `xterm-256color-italic`.
3. Restart iTerm.

This [blog post][] has more information.
